# CyberPower

**CyberPower PDU by SNMP.yaml** - Шаблон мониторинга PDU CyberPower ePDU2. Тестировался на модели PDU83401.

Конфигурация Daisy Chain в шаблоне не поддерживается. Расположение сенсора климата захардкодил. Исходя из показаний датчиков температуры и влажности считается точка росы.
